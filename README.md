# pulse-flow

A pulseaudio configuration tool

![](demo.png)

### Dependencies
- [libgtkflow](https://github.com/grindhold/libgtkflow)
- [meson (build)](http://mesonbuild.com)

### Run
```
$ git clone --recursive https://github.com/benwaffle/pulse-flow.git
$ cd pulse-flow
$ mkdir build
$ meson build
$ ninja -C build
$ build/pulse-flow
```
